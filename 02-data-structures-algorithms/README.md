# Module 02 — Data Structures & Algorithms

This module covers fundamental data structures and algorithm concepts through hands-on,
scenario-based exercises simulating real-world applications.

## Exercises

| # | Exercise | Key Concepts | Data Structure / Algorithm |
|---|----------|-------------|---------------------------|
| 01 | [Inventory Management System](./exercise-01-inventory-management-system/) | CRUD operations, Hashing | HashMap |
| 02 | [E-Commerce Search](./exercise-02-ecommerce-search/) | Asymptotic notation, Search | Linear Search, Binary Search |
| 03 | [Sorting Customer Orders](./exercise-03-sorting-customer-orders/) | Sorting algorithms | Bubble Sort, Quick Sort |
| 04 | [Employee Management System](./exercise-04-employee-management-system/) | Array representation | Arrays |
| 05 | [Task Management System](./exercise-05-task-management-system/) | Linked data structures | Singly Linked List |
| 06 | [Library Management System](./exercise-06-library-management-system/) | Search algorithms | Linear Search, Binary Search |
| 07 | [Financial Forecasting](./exercise-07-financial-forecasting/) | Recursion, Optimization | Recursion, Memoization |

## How to Run Any Exercise

```bash
cd exercise-XX-name/src
javac model/*.java service/*.java Main.java
java Main
```
